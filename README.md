# League Portal - Premium Edition

This is a fully data-driven, real-time cricket league portal built with React and Firebase Firestore.

## Features
- **Real-Time Bidding**: synced across all users.
- **Admin Dashboard**: Manage players, teams, and match info.
- **Premium Design**: Full glassmorphism and interactive mouse animations.
- **Role-Based Login**: Secured access for Admin and Regular Users.

## Tech Stack
- React.js
- Firebase (Firestore & Auth)
- CSS3 (Premium Glassmorphism)
- Vite
